# Welcome to Bowser Gallery Documentation

Here you can find any useful in depth information and documentation concerning Bowser Gallery. **This documentation is currently in progress**

**What is Bowser Gallery?**

Bowser Gallery is a fan made Discord Application (also known as Bot), that allows you via a simple command to get a random picture of Bowser, a character of the *"Super Mario"* franchise.

This Application is owned and managed by the owner of this repository. No public code is available.

This Application can be installed on your Discord server [here](https://discord.com/discovery/applications/1244618426228674642)
