# Bowser Gallery Request Guidelines

Here are some guidelines concerning picture requests if you are making some with Bowser Gallery.

## File type and size

- File type must be on a popular format such as: **jpg (jpeg), png (webp) or gif.** Your image extension is different than those? Feel free to use an online image converter!
- File size must be equal or lower than 4 MB.
- There's no dimension restriction.

## File content

- Must be Bowser, ideally him alone on the picture. Bowser and Bowser Jr. / Koopalings together are fine as well.
- Must be SFW. (Safe for Work)
- AI generated content is not accepted.
- Fan arts are not accepted at this time, even with artist credit/permission because art repost is considered as stealing.

## Other notes

- If your request is accepted, you'll get a Direct Message from the bot saying it's added. (Unless your Direct Messages are off)
- Request review time can vary depending of multiple factors but in general it's taking from a **couple hours to a couple days.** Please do not send the same request over and over again until your request has been reviewed.
- You can follow the status of your request at any time with the ``/viewreq`` command! 
- Requests are accepted / declined on my own discretion, decline reasons can be found with the ``/viewreq`` and the request ID you get when you sent the request. **All requests declined have a reason.**
- If you are at any point spamming the command or sending unrelated pictures to troll, you may be limited temporarily or permanenetely from making requests in the future.
- If you have any questions, feel free to [contact me](https://r0mainmk.bowserhub.net/socials)!
