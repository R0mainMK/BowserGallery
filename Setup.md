# How to install Bowser Gallery on my server?

- In the Discord App Directory you can search by typing "Bowser" and normally it cames as the first result, or you can go [here](https://discord.com/discovery/applications/1244618426228674642)
- Click on "Add App"
- You are asked to choose a server to install the bot, please note that you need the *"Manage Server"* permission, or the *"Administrator"* permission or being the owner of the server to add the bot to the server you want.
- You are now showing the permissions that the bot is seeking for, change nothing and click on "Authorize"
- And that's it! The bot is installed. Have fun!

  ## Frequentely Asked Questions

**"I want the bot to access my private channels"**
  
You can do so by adding the "Bowser Gallery" role on the "Who can access?" list  on the channels settings. It's strongly **discouraged** to grant the bot the "Administrator" permission just to allow the bot to access all your server channels because if bot tokens are getting compromised by an attack, attackers can nuke your server.

**"My members can't access the commands"**

Be sure that the members have the *"Use Application Commands"* permission, if you want everyone on your server to use the bot, then allow this permission on the @everyone role so even if other roles doens't have the permission enabled, they'll still able to use the bot.

**"Nothing happens when a command is triggered"**

That means you probably disallowed the bot a required permission (such as *"Send Messages"* or *"View channels"* for example), you can fix this by either checking the permissions of the "Bowser Gallery" role or by removing the bot from your server and adding it back. If the problem still persists, you can contact me or opening an issue here.

**Are messages or anything in my server logged?"**

Messages sent in your server are **NOT** logged at all, you can check what data is collected from the bot on [this page](https://bowser.gallery/privacy.html)
